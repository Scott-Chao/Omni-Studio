# Smart-Markdown 功能规格文档

## 一、核心编辑器层 (The Core Editor)

整个应用的基础，重点在于流畅的编辑体验与标准的 Markdown 渲染。

- **多模式编辑器**：支持源码模式、实时预览模式以及“WYSIWYG（所见即所得）”混合模式。
- **语法高亮引擎**：针对 C/C++、Python、Java 等语言提供深度语法高亮。
- **数学公式渲染**：集成 LaTeX 渲染引擎（如 KaTeX），方便记录程设中的数学推导（如复杂度分析、概率论）。
- **多文档界面 (MDI)**：支持标签页切换，允许同时打开多个笔记进行对比参考。

## 二、局部评测系统 (Local Judge Module)

参考 NeverJudge 并进行升级，将笔记变为一个单机 OJ。

- **评测配置挂载**：在代码块上方显示悬浮工具栏，允许用户关联对应的 `.in`/`.out` 测试用例文件夹。
- **沙盒执行**：
  - **资源限制**：监控 QProcess 的运行时间与内存占用，若超过设定的阈值则返回 Time Limit Exceeded 或 Memory Limit Exceeded。
  - **输入重定向**：自动将测试用例喂给程序，并截获标准输出进行文本比对。
- **差异化对比 (Diff View)**：当结果为 Wrong Answer 时，弹出侧边栏对比“预期输出”与“实际输出”的差异。

## 三、算法可视化插件 (Algorithm Visualization)

将抽象的逻辑具象化，这是“元学习”的核心亮点。

- **Mermaid 图表集成**：在 Markdown 中直接书写脚本，实时渲染流程图、时序图、甘特图。
- **内核动态追踪 (Kernel Trace)**：
  - **变量监测**：通过特殊的语法注释（如 `//@viz:arr[1..10]`），程序在运行时会提取特定变量的值。
  - **图形化呈现**：利用 QGraphicsView 实时画出数组的交换、二叉树的旋转、堆栈的进出等动画。
- **双向通信**：使用 QWebChannel 实现 C++ 运行后端与 Web 预览区（D3.js 或 ECharts）的数据交互。

## 四、知识网络与第二大脑 (Second Brain & Backlinks)

借鉴 Obsidian 的理念，解决学习碎片化问题。

- **双向链接**：支持 `[[文件名]]` 语法，自动建立笔记间的跳转关系，并在侧边栏展示“哪些笔记引用了当前内容”。
- **动态知识图谱**：基于 QGraphicsScene 绘制所有笔记的关联拓扑图，节点大小可根据笔记字数或引用频次动态调整。
- **全局搜索与索引**：支持正则表达式全局搜索，并能对代码块中的特定函数名进行全文索引。

## 五、记忆强化与复习 (Spaced Repetition)

利用艾宾浩斯遗忘曲线，将笔记转化为长久记忆。

- **闪卡语法**：识别特定标记（如 `> [!QUESTION]`），将其内容隐藏并生成“复习按钮”。
- **复习看板**：根据用户的点击反馈（记得/模糊/忘记），将该知识点加入本地 SQLite 数据库进行排期复习。
- **系统通知提醒**：即使软件未完全打开，也可通过系统托盘提醒用户今日待复习的内容。

## 六、协作与工程化 (Collaboration)

- **GitHub 深度集成**：内置简单的 Git 命令面板，支持一键 Commit & Push，方便组员间通过 GitHub 进行代码协作。
- **版本历史**：记录笔记的修改历史，防止误删代码逻辑。
- **导出功能**：支持将带代码评测结果的笔记导出为 PDF 或带交互功能的 HTML 页面。
