# Smart-Markdown 功能文档

## 一、核心编辑器层 (The Core Editor)

- **多模式编辑器**：支持源码编辑、实时预览、分屏预览、代码编辑、PDF 阅读、SMD 单元格编辑。
- **语法高亮引擎**：针对 C/C++、Python 提供深度语法高亮。
- **数学公式渲染**：集成 KaTeX 渲染 LaTeX 公式。
- **Mermaid 图表**：在 Markdown 中直接绘制流程图、时序图、甘特图。
- **多文档界面 (MDI)**：支持标签页切换，等宽/非等宽两种模式。

## 二、代码智能 (Code Intelligence)

- **LSP 代码补全**：C++ 通过 clangd，Python 通过 Jedi。
- **悬停提示**：类型信息和函数签名实时显示。
- **诊断波浪线**：错误/警告下划线，底部面板展示。

## 三、SMD 单元格编辑器 (SMD Cell Editor)

- **单元格格式**：类似 Jupyter Notebook，Markdown/C++/Python 三种类型。
- **双模式系统**：命令模式（键盘导航）与编辑模式（文本编辑）。
- **单元格执行**：Markdown 渲染为图片，C++ 编译运行，Python 持久化进程。
- **双向转换**：一键 `.md` ↔ `.smd` 转换。

## 四、知识网络 (Knowledge Network)

- **双向链接**：支持 `[[文件名]]` 语法，自动建立笔记间跳转关系。
- **反向链接面板**：展示当前文件的引用来源。
- **#tag 系统**：标签标记、导航与关联文件展示。
- **全文搜索**：在当前目录所有文本文件中检索关键词。
- **大纲导航**：自动解析标题，点击跳转。

## 五、编译运行与评测 (Compile & Judge)

- **编译运行**：C/C++ 编译后运行，Python 直接运行。
- **本地评测**：选择测试用例批量运行，OJ 风格结果与对比。
- **错题本**：自动记录失败用例。
- **OpenJudge 集成**：题目浏览、登录管理、代码提交、IDE 模式。

## 六、UI 与交互 (UI & Interaction)

- **无边框窗口**：自定义标题栏，拖拽移动与缩放。
- **文件树**：面包屑路径、内联新建/重命名/删除、拖拽移动。
- **主题系统**：深色/浅色双主题，实时切换。
- **设置面板**：悬浮遮罩面板，分类配置。
- **AI 助手**：多轮对话，流式输出。
